## Quick File Lookups

**Need to...**

Add a new page?
→ Create in `pages/`, add route in `App.tsx`, link in `Sidebar.tsx`

Add API method?
→ Add to `services/api.ts`, create hooks in `hooks/useQueries.ts`

Create a form?
→ Use `React Hook Form + Zod` pattern from `features/`

Style something?
→ Use Tailwind classes or edit `styles/globals.css`

Add a component?
→ Place in `components/` folder, export from index if needed

Debug API?
→ Check `services/api.ts` interceptors and network tab

Find types?
→ Check `types/index.ts` for all type definitions
